# raiseradar-site

Landing page for **RaiseRadar** — a weekly list of US companies that just closed a private funding round, sourced from SEC EDGAR Form D filings.

- `index.html` — the site (static, single page)
- `sample.csv` — sample of the weekly feed (served as the signup lead magnet)
- [`docs/COMMERCIAL_AUDIT.md`](docs/COMMERCIAL_AUDIT.md) — commercial audit: assets, data quality, competition, where the value is
- [`docs/GTM_STRATEGY.md`](docs/GTM_STRATEGY.md) — go-to-market strategy: positioning, ICP, pricing, channels, 30/60/90 plan
